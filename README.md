# Full-Stack Vehicle Marketplace

A web-based vehicle sales platform built with ASP.NET Core and SQL Server.

## Features
- User authentication with role-based access control
- Vehicle listing and management system
- Reservation booking with conflict prevention
- Admin dashboard with management capabilities
- Dynamic search and filtering using AJAX

## Tech Stack
- C#
- ASP.NET Core
- SQL Server
- HTML5, CSS, JavaScript, jQuery

## My Role
Led a team of 4 students, designed the database schema, implemented authentication extensions, and developed core business logic.
